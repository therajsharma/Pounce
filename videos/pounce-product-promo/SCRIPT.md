# Script

## Narration

Dependency changes shouldn't quietly become repo state.

Pounce checks exact releases before the install, watches the shell while agents work, and keeps every allow, warn, and block verdict attached to the same workspace.

Feed trust stays visible.

Risky packages stop before they settle.

## Notes

- Target duration: 20 seconds.
- Spoken pace: roughly 2.3 words per second with short pauses.
- CTA is implied visually through the final dashboard state and install command.
