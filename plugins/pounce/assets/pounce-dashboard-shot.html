hook timeline

Three stages keep the dependency story attached to the agent turn.

1
UserPromptSubmit Snapshot dependency manifests and lockfiles before tools run.
2
PreToolUse Inspect install commands and block known bad or non-exact specs.
3
Stop Diff dependency files against the snapshot and flag unexplained edits.
